# 프로젝트 설명 페이지

```
- 미션 생각나는 거 작성

- 회원 등록 페이지에서 휴대폰 번호는 숫자만 입력, 이름은 영어,숫자,한글만 사용하게 입력

- 모든 회원가입 페이지에서 마지막에 한번 더 확인하는 안내 메시지 설정

- 제일 마지막에 할 미션 -
현재 창을 종료하면, DB Connection을 종료시키는 함수도 구현해야 함


```

## 설명

1. utils -> DB 초기 셋팅
2. prop -> DB 연결 셋팅
3. img -> ico 파일
4. fxml -> fxml 파일
5. controller -> Controller 파일
6. css -> css 파일
